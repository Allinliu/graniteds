Gas3 @version@ (AS3 code generation) Plug-In for Eclipse.


 
  1. Installation:
     
    Unzip org.granite.gas3_@version@.zip into the plugins directory of your Eclipse distribution (this will create an org.granite.gas3_@version@ directory).
     
  2. Documentation:
     
    See documentation on Granite Data Services main site.
