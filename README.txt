
Granite Data Services Android client (http://www.graniteds.org)
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------

* Provide:

  An Android client library and framework that may be used to connect to a
  remote GraniteDS server.

* Requirements:

  JDK 6+, Android 4+
  
* Build:

  Download the complete source bundle and import it as an Eclipse project.
  Building the jar requires Gradle: 
  
  gradle build
