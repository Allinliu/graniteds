
Granite Data Services Builder (http://www.graniteds.org)
--------------------------------------------------------------

* Provide:

  - the "gas3" Ant task, packaged as an Eclipse Ant task (may be used in any build.xml
    under Eclipse without taskdef declaration).
  - the granite Eclipse builder, that may be activated via the "Add GraniteDS Nature"
    entry in the project contextual menu.

* Requirements:

  Eclipse 3.2+, JDK 6+.

* Installation:

  Close Eclipse, remove any older version of the plugin, drop the new
  org.granite.builder_x.x.x.jar into your <ECLIPSE_HOME>/plugins directory,
  then restart Eclipse.
