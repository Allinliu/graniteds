
Granite Data Services JavaFX client (http://www.graniteds.org)
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------

* Provide:

  A JavaFX client library and framework that may be used to connect to a
  remote GraniteDS server.

* Requirements:

  JDK 6+, JavaFX 2.2+
  or JDK 7+ (includes JavaFX)
  
* Build:

  Download the complete source bundle and import it as an Eclipse project.
  Building the jar requires Gradle: 
  
  gradle build
